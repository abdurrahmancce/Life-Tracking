# 📊 Life Analytics Dashboard

A modern and interactive **Life Analytics Dashboard** built with **HTML, CSS, JavaScript, Chart.js, and Local Storage**. This application helps users track daily activities, analyze productivity, monitor habits, set goals, and gain meaningful insights from their time usage.

---

## 🚀 Overview

Life Analytics Dashboard transforms daily activities into actionable insights. It allows users to log how they spend their time, visualize activity patterns, track productivity trends, monitor personal goals, and build better habits through data-driven decision making.

The application works entirely in the browser and stores all data locally, ensuring privacy and offline accessibility.
